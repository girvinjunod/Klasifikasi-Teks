# Klasifikasi-Teks
Tugas Klasifikasi Teks dari mata kuliah IF4072 NLP

# Prerequisites
- Python 3.8 (minimal)
- pip
- pipenv

# Dependencies
- sklearn
- ipykernel
- xgboost
- pandas
- gensim
- numpy
- tensorflow
- keras
- matplotlib
- seaborn
- emoji
- transformers

# Installation
1. Clone this repository.
2. Install the dependecies with `pipenv install`
3. You can run each model inside the `src` folder from the `.ipynb` file.
